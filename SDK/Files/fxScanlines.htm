fxScanlines

Displays scanlined bitmap, first stretching or compressing it to fit the dimensions of the destination rectangle, if necessary.

If the function succeeds, the return value is 1 (true); if fails, the return value is 0 (false).

Specifies wether horizontal (0) or vertical (1) scanlines will be drawn
